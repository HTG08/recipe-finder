# Recipe Finder App

A simple and interactive React application that allows users to search for recipes based on ingredients and explore detailed cooking instructions.


## Live Demo

-- https://recipe-finder-beryl-two.vercel.app/


## Features

* Search recipes by meal name
* Display recipe images, titles and procees 
* Toggle to view detailed cooking instructions
* Add and remove recipes to favorites
* View total favorites count
* Filter vegetarian recipes



## Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS
* TheMealDB API



## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/recipe-finder.git
cd recipe-finder
npm install
npm run dev
```
